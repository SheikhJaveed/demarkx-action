# DeMarkX Markdown Fixer

> Deterministic Markdown repair for CI and teams.

DeMarkX automatically repairs broken Markdown syntax in pull requests and commits safe fixes back to your branch. Unlike traditional linters, it doesn’t just complain — it heals.

---

## What It Does

- Repairs missing spaces in headers and lists  
- Fixes broken fenced code blocks  
- Standardizes Markdown structure  
- Normalizes numbered and bullet lists  
- Detects structural issues  
- Deterministic and idempotent output
